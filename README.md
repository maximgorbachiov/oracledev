# Tarot Service API

A backend API service for tarot card readings and user management, built with ASP.NET Core and Entity Framework.

## Features

- **User Management**: Register, login, and manage user accounts
- **Card of the Day**: Get daily tarot card readings
- **Database Support**: PostgreSQL (development) and Oracle DB (production)
- **Deployment**: Configured for Docker Compose and Oracle Cloud Infrastructure (OCI)

## Technology Stack

- .NET 8.0
- Entity Framework Core
- PostgreSQL / Oracle Database
- Nginx (reverse proxy)
- Docker & Docker Compose
